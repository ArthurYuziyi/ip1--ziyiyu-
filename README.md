# Automation: The Game - Group F

This is the repository for the Automation: The Game project, developed as part of the COSI 103A course at Brandeis University.

## Project Overview

A deck-building card game prototype inspired by Dominion mechanics. Players build their decks by purchasing cards from a shared supply, with the goal of accumulating Automation Points (AP).

## Subprojects

- **automation-the-game/**: Complete game implementation
- **ip1/**: Individual Project 1 assignment
